The Basic Language of the Web HTML

Posted by Developer Cat on Monday, December 06th 2021

All modern websites and web applications are build using three fundametal technilogies: HTML,CSS and JavaScript. These are the languages of the web in this post, let's focus on HTML. We will learn what HTML is all about and why you should learn it.

What is HTML

HTML stands for HyperText Markup Language. It's markup language that web developers use to structure and describe the content od webpage (not a programming language)

HTML consists of elements that describe different type of content: paragraphs, links. headings, images, video, etc. Web browsers understand HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at the MDN Web Docs
